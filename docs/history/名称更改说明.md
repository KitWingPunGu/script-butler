# 扩展名称更改说明

## 📝 更改内容

### 原名称
**NPM 脚本管理器** (npm-script-manager)

### 新名称
**脚本管家** (Script Butler / script-butler)

---

## 🎯 更改原因

### 为什么选择"脚本管家"？

1. **更专业** ⭐⭐⭐⭐⭐
   - "管家"一词体现全方位服务
   - 适合企业和个人用户
   - 专业且亲切

2. **更准确** ⭐⭐⭐⭐⭐
   - 准确反映功能：管理 + 执行
   - 不局限于 NPM，支持多种包管理器
   - 强调"智能管理"的核心价值

3. **更易记** ⭐⭐⭐⭐⭐
   - 简洁有力（3个字）
   - 形象生动，容易联想
   - 朗朗上口

4. **更友好** ⭐⭐⭐⭐
   - 搜索友好度高
   - 适合 VS Code 市场展示
   - 国际化友好（Script Butler）
