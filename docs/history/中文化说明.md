## 📝 注意事项

### 保持英文的部分

以下内容保持英文，因为它们是技术标识符：

1. **命令 ID**
   - `npmScriptManager.refresh`
   - `npmScriptManager.runScript`
   - 等等...

2. **视图 ID**
   - `npmScripts`
   - `npmFavorites`

3. **扩展 ID**
   - `npm-script-manager`

4. **文件路径和技术配置**
   - 所有代码中的变量名、函数名
   - TypeScript 类型定义
   - 配置文件路径

### 终端命令

终端中执行的命令保持英文：
```bash
npm run dev
npm run build
```

这是因为这些是 npm 的标准命令，不应该翻译。
